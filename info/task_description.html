Enunciado

Escreva uma função que receba uma lista de preços diários de uma ação e retorne True se o preço da ação subiu por mais de 3 dias consecutivos, e False caso contrário.

Entrada: Uma lista de inteiros.

Saída: Um valor booleano (True ou False).
